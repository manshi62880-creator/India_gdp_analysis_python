# Indian States GDP Analysis using Python

This project explores the economic performance of Indian states by comparing:

1. Total GDP
2. Per Capita GDP

The goal is to understand how economic size and individual prosperity can differ across states.

## Dataset
State-wise GDP dataset (Kaggle)

## Tools Used
- Python
- Pandas
- Matplotlib

## Analysis Performed
1. Data loading and cleaning
2. GDP ranking of states
3. Per-capita GDP calculation
4. Data visualization

## Key Insights
- Maharashtra leads in total GDP among selected states.
- Telangana shows strong performance in per-capita GDP.
- Large population states may have high total GDP but lower per-person income.

## Visualization
The project includes comparison charts showing:
- Total GDP by state
- Per-capita GDP by state
